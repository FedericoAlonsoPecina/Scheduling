All the instances of this Folder have the name E_X_N, where X is the number of the instance and N is the number of jobs in the instance

Inside, all the files have the format:

#number of jobs (n)

#release_time_1  #process_time_1  #duedate_1
#release_time_2  #process_time_2  #duedate_2
...
#release_time_n  #process_time_n  #duedate_n
